# ProjectResult

The result model for project information.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| id | integer | Project ID |
| name | string | Project name |
| summary | string | Project description |
| tokens | TokenResult[] | Array of token information |
| tweet_summaries | TweetSummary[] | Array of tweet summaries |
| twitter_id | string | Twitter account ID |
